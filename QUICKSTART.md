# Quick Start

## Install & Run

```bash
npm install
npm run dev
```

Visit: http://localhost:3000

## First Use

1. Enter username
2. Toggle 7-week storage if needed
3. Save

## Create Room

Click "Create Room" → Copy code → Share

## Join Room

Paste code → Click "Join"

## Features

- **Edit**: Click "Edit" on your messages
- **History**: Click "Show original" on edited messages  
- **Retention**: Click "24H"/"7W" badge to toggle
- **Copy Code**: Click room code in header
- **Settings**: Click "Settings" on home

## Tips

- Recent rooms shown on home page
- Typing indicator shows automatically
- Online users listed in header
- Messages auto-expire based on retention

## Shortcuts

- `Enter` - Send / Save edit
- `Esc` - Cancel edit

## Production

```bash
npm run build
npm start
```

Server runs on port 3000 by default.
